# Gayatri Testing

This is a test project for Playwright and other testing concepts.

## Project Structure

- `Chapter_01_Basics`: Contains basic examples.
- `chaper_02_JAVA_Concepts`: Contains java concept examples.
- `Test.js`: JavaScript test file.
- `Ashok.java`, `Palin.java`: Java source files.

## Getting Started

1. Clone the repository.
2. Open the project in your preferred IDE (VS Code, IntelliJ, etc.).
3. Run the tests or scripts as needed.
